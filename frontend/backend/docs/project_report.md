# FraudShield AI

## Problem

Millions of citizens lose money through
digital fraud and scam activities.

## Solution

FraudShield AI detects scam messages,
fraudulent communications and provides
real-time risk assessment.

## Features

- Scam Detection
- Fraud Risk Score
- Citizen Protection
- AI Recommendations

## Tech Stack

- Python
- Flask
- HTML
- CSS
- JavaScript

## Future Scope

- Voice Scam Detection
- Deepfake Detection
- Fraud Network Analysis
